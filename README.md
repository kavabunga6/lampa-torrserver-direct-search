# Lampa TorrServer Plugins

> Статус: WIP. В репозитории лежат два экспериментальных плагина для Lampa. Они уже пригодны для тестирования, но поведение Lampa API и внутренних экранов может отличаться между сборками, поэтому возможны поломки и изменения настроек.

## Плагины

### TorrServer Direct Search

Файл: `torrserver-direct-search.js`

Добавляет поиск контента напрямую через TorrServer, без обязательного поиска через TMDB или CUB.

Что делает:

- добавляет кнопку меню `TS поиск`;
- добавляет источник `TorrServer` в общий поиск Lampa, если это включено в настройках;
- ищет через TorrServer endpoint `/torznab/search/?query=...`;
- умеет fallback на `/search/?query=...`;
- открывает выбранный результат через штатную страницу `full` Lampa;
- запускает выбранный torrent через штатный `Lampa.Torrent.start`;
- по возможности использует стандартные карточки, заглушки, full-экран и модалки Lampa.

URL установки:

```text
https://kavabunga6.github.io/lampa-torrserver-direct-search/torrserver-direct-search.js
```

### Search Source Filter

Файл: `search-source-filter.js`

Добавляет настройки для включения и отключения источников общего поиска Lampa.

Что делает:

- позволяет отключать любые найденные источники поиска;
- по умолчанию отключает `CUB` и `AI-ассистент`;
- не удаляет источники навсегда, их можно включить обратно в настройках;
- запоминает дополнительные источники, которые встретил при открытии поиска;
- предотвращает запросы к отключённым источникам.

URL установки:

```text
https://kavabunga6.github.io/lampa-torrserver-direct-search/search-source-filter.js
```

## Установка

1. Откройте в Lampa `Настройки -> Расширения -> Добавить плагин`.
2. Добавьте URL нужного плагина.
3. Если нужны оба сценария, добавьте оба URL по очереди.
4. После обновления файла на GitHub Pages можно добавить cache-bust параметр, например `?v=23`.

Пример:

```text
https://kavabunga6.github.io/lampa-torrserver-direct-search/torrserver-direct-search.js?v=23
```

## Настройки TorrServer Direct Search

Раздел: `Настройки -> TorrServer поиск`

- `Использовать TorrServer из настроек Lampa` - берёт адрес из стандартных настроек TorrServer в Lampa.
- `Адрес TorrServer` - ручной адрес, например `http://192.168.1.10:8090`.
- `Добавить вкладку в общий поиск` - включает источник `TorrServer` в обычном поиске Lampa.
- `Таймаут поиска, секунд` - сколько ждать ответ TorrServer.
- `Источник поиска TorrServer` - `Torznab`, `Rutor` или `Torznab + Rutor`.
- `Искать постеры через TMDB` - опциональная подгрузка реальных постеров. По умолчанию выключена.
- `Сколько постеров искать` - лимит дополнительных TMDB-запросов на один поиск.
- `Лимит результатов` - сколько результатов TorrServer показывать в поиске.

## Настройки Search Source Filter

Раздел: `Настройки -> Источники поиска`

- фильтр можно полностью включить или выключить;
- каждый найденный источник можно включить или отключить отдельно;
- новые источники появляются в настройках после того, как Lampa хотя бы раз открыла поиск и плагин их увидел.

## Совместное использование

Плагины независимые:

- `torrserver-direct-search.js` отвечает за прямой поиск через TorrServer;
- `search-source-filter.js` отвечает только за фильтрацию источников общего поиска.

Их можно ставить отдельно. Если поставить оба, можно оставить в общем поиске только нужные источники, например `TMDB` и `TorrServer`, а `CUB` и `AI-ассистент` отключить.

Кнопка `TS поиск` всегда открывает поиск только по TorrServer и не запускает TMDB/CUB параллельно.

## Проверка

```bash
npm test
npm run test:smoke
```

Smoke-тест по умолчанию проверяет:

```text
http://torrserver.home/torznab/search/?query=matrix
```

Адрес и запрос можно заменить:

```bash
TORRSERVER_URL=http://torrserver.home TORRSERVER_QUERY=matrix npm run test:smoke
```

## Примечания

- GitHub Pages для публичного репозитория обычно можно использовать как простой хостинг JS-файлов.
- В Lampa иногда нужно менять `?v=...`, чтобы сбросить кеш плагина.
- Плагин старается использовать штатные экраны Lampa, но часть адаптерного кода всё ещё нужна, потому что TorrServer отдаёт не TMDB/CUB-карточки, а torrent-результаты.
